# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/0b74d0cb-1918-4104-bf66-854f657e4fff

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Optionally set `VITE_POSTHOG_KEY` and `VITE_POSTHOG_HOST` if you want analytics
3. Run the app:
   `npm run dev`
